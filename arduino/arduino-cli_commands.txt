List of Arduino-CLI Commands
2023.dec15 fri 1210mf

FREQUENTLY USED COMMANDS
* arduino-cli compile --fqbn arduino:avr:uno
* arduino-cli upload --port /dev/cu.usbserial-11230 --fqbn arduino:avr:uno
* arduino-cli board list
* arduino-cli monitor --port /dev/cu.usbserial-11230

MORE COMMANDS
* arduino-cli core install arduino:avr

LEGENDS:
* -b or --fqbn: fully qualified board name
* -p or --port: port
